# Images Directory

This directory should contain the following image files:

## Required Images:
- logo.png (120x120px) - App logo for about page
- share-prediction.png (400x300px) - Default image for sharing predictions
- default-avatar.png (80x80px) - Default user avatar
- empty-state.png (200x150px) - Empty state illustration
- loading.gif (40x40px) - Loading animation

## App Icon:
- icon.png (1024x1024px) - App icon source file
- Various sizes will be generated automatically by uni-app

## Splash Screen:
- splash.png (750x1334px) - Launch screen image

## Background Images:
- bg-gradient.png - Gradient background for certain pages
- bg-pattern.png - Pattern background if needed

## Design Guidelines:
- Use PNG format for images with transparency
- Use JPG format for photos and complex images
- Optimize file sizes for mobile
- Provide @2x and @3x versions for high-DPI displays
- Use consistent color scheme matching app theme

## Color Scheme:
- Primary: #d32f2f (Red)
- Secondary: #666666 (Gray)
- Success: #4caf50 (Green)
- Warning: #ff9800 (Orange)
- Error: #f44336 (Red)
- Background: #f5f5f5 (Light Gray)
