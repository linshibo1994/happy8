# Tab Bar Icons

This directory should contain the following icon files:

## Required Icons:
- home.png (32x32px) - 首页图标 (未选中状态)
- home-active.png (32x32px) - 首页图标 (选中状态)
- predict.png (32x32px) - 预测图标 (未选中状态)  
- predict-active.png (32x32px) - 预测图标 (选中状态)
- history.png (32x32px) - 历史图标 (未选中状态)
- history-active.png (32x32px) - 历史图标 (选中状态)
- member.png (32x32px) - 会员图标 (未选中状态)
- member-active.png (32x32px) - 会员图标 (选中状态)
- profile.png (32x32px) - 个人图标 (未选中状态)
- profile-active.png (32x32px) - 个人图标 (选中状态)

## Design Guidelines:
- All icons should be in PNG format
- Normal state: Gray color (#999999)
- Active state: Primary color (#d32f2f)
- Size: 32x32px for @1x, also provide @2x and @3x versions
- Background: Transparent
- Style: Simple, clean, recognizable

## Icon Sources:
You can use icon libraries like:
- Iconfont (iconfont.cn)
- IconFont (iconfont.cn)
- Material Icons
- Ant Design Icons
- Or create custom icons

## Naming Convention:
- Use lowercase with hyphens
- Include state suffix (-active for selected state)
- Use descriptive names
